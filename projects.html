Omnivision

IP Camera manager

Basic info:

Created on:
2014.

Technology:
C#, AForge.NET

Software:
Visual Studio, Adobe Photoshop

Project:

Github link »


Details:

Omnivision is a simple IP camera manager written using C# and AForge.NET addon for IP cameras, image-filtering, video and computer vision. Users can add their IP camera details (name, IP address, username and password) which are being stored locally. Each camera can be ran in separate window with multiple instances using multi-threading with minimal impact on latency (dependant on Internet connection).

Hexagonal Melody - Drums

It's a Simon says game... but with drums!

Basic info:

Created on:
2014.

Technology:
C#, SharpDX, MySQL .NET Connector

Software:
Visual Studio, Adobe Photoshop, Audacity, XAMPP

Project:

Github link »


Details:

Hexagonal Melody is a simple "Simon Says" type of game where you have to follow and repeat a random drum pattern and score as much points before you make a mistake. It features 3 sets of drums which alters the game difficulty. Game features a Simon Says mode and free-play mode where you can play the drums using mouse and keyboard. "Simon says" mode features difficulty modes and a simple scoring system. There is a feature to upload highscore, but it's not currently active due to the database not being available.

Simple image to ASCII art converter with text editing based on image editing

Basic info:

Created on:
2014.

Technology:
C#

Software:
Visual Studio

Project:

Github link »


Details:

This project was made as a basis for my image-manipulation and editing studies. Goal was do make a lightweight, fast and robust application that converts any submited image and displays it with user-selected editing options. Application enables fetching images from an IP camera in succession (primitive video) with minimal latency.

2014.

Technology:

Details:

I was thinking outside of the box with this project. I've used button controllers to create makeshift graphics and I've used free available chiptune music as a soundtrack. Game is an "endless runner" and you have to jump over the pits and obstacles, shoot enemies and smash the doors in your way. The game features various difficulty modes (speeds), variable color schemes, randomly generated obstacles and a scoring system.
